Credit-LLM ⊧ Credit Governor -- a governed consumer/commercial credit-granting actor. This deploy exposes an intake-only, read-only edge API (/api/loan/*) -- :jurisdiction/assess, :creditworthiness/screen, :loan/approve and :loan/disburse are NEVER exposed over HTTP, by design, to preserve this actor's independent governance. See the source repository for the full README/ADR.
